PROBLEM-STATEMENTS

There are 2 problem statements named GOLD & SILVER.

GOLD PROBLEM STATEMENT

Develop an innovative AI based news aggregator application customized explicitly for strategic & geopolitical significance (Defence centric).

Technical Details

Participants are required to design and implement an end to end application which delivers to the following technical pointers:

  1. AI based web/mobile news aggregator application trained specifically for strategic content
  2. The system should have the capability to learn from the users likes and interests and pick news from user's specific preferential domestic or foreign based online news articles.
  3. The application should provide the user with an input field to include their interests, sub domains and specific keywords, which is used and checked according to relevance across the different sites. NLP techniques can be used to analyze and understand the content of the articles, which includes identifying key topics, keywords, entities, topic modeling etc. to generate concise, coherent, fluent and relevant summary (150 words or less) of the news article along with a word cloud, topic modeling and sentiment analysis in that context.

Technical Deliverables

  • Data collection through web scraping, followed by aggregation to collate a Defence centric database repository of newsfeed articles/RSS feeds containing latest and relevant information explicitly in strategic and geopolitical context [should include both domestic and foreign sources without any bias].
  • Using NLP based techniques to understand the user's keywords and query (input). It should have capability to analyze and understand the context and content of the database and deliver the most relevant insights pertaining to the user requirements.
  • Machine Learning algorithms and Recommendation system to cater to user profiling for personal preferences and interests integrated with the recommendation engine to deliver those insights.
  • An user interaction system to refine recommendations and functionality to provide feedback.
  • A functional UI in form of application (webapp and mobile app) integrated with the backend to fullfill the deliverables and enhance user's overall experience.

SILVER PROBLEM STATEMENT

Develop an innovative text summarization tool that can efficiently process and summarize handwritten texts, all while operating offline.

Technical Details

Participants are required to design and implement a text summarization tool that meets the following technical specifications :

  1. Handwriting Recognition: Implement an OCR (Optical Character Recognition) system capable of accurately converting handwritten text into digital text. The system should handle diverse handwriting styles and formats.
  2. Summarization: Develop a summarization algorithm using a NLP algorithm or language model that can generate concise and contextually relevant summaries from the digitized text.
  3. Offline Functionality: Ensure the entire process, from OCR to summarization, operates offline to maintain privacy and accessibility in areas with no internet connectivity.
  4. Resource Efficiency: Optimize the tool to run on local devices with min. RAM 16 GB and on portable systems.

Technical Deliverables

  • Accuracy: The OCR system should have high accuracy in recognizing various handwriting styles.
  • Coherence: Summaries should be coherent, retaining the essential information and context of the original text.
  • Efficiency: The tool should be lightweight and efficient, with minimal latency in processing and summarizing text. User Interface: Develop a user-friendly interface that allows easy input of handwritten texts and displays the summarized output clearly.
  • Language Support: Only English File Formats: Multimodal (images, pdf, text, word, excel documents, json, xml etc.)

RULES

  1. Neurohack is an online multi-stage event that will be conducted for 2-3 days.
  2. There is no restriction on its participation; participants can participate from anywhere in the world.
  3. Two students from different colleges can be a part of the same team. However, one person cannot be a part of multiple teams for the same event.
  4. Decisions taken by the judges and organizers will be considered final and binding to all.
  5. Teams who do not register will not be eligible for prizes, and their solution will not be taken under consideration.
  6. Participants will have to share their code.
  7. Plagiarized solutions will not be considered. Teams will be disqualified if found in any plagiarism check.
  8. Rules and criteria may be changed without any prior intimation. Participants are requested to check the MindSpark regularly for updates.
